# Project Instructions for Claude

## Git Workflow
- After completing any task that modifies code files, always commit and push changes to git
- Write clear, descriptive commit messages summarizing the changes

## Project Purpose
- ToyAyPricecompare: Price comparison tool for Toyota Aygo lease deals

## Project Structure
- Main code goes in project root or `src/` directory
- Test files follow pattern `test_*.py`
- Output files go in `output/` directory (gitignored)
